'Intel? Of toch AMD?..'

Als je ooit CPU's heb moeten vergelijken heb je je deze vraag vast wel afgevraagd.

Je wilt natuurlijk de beste, mooiste, maar boven alles, de snelste CPU hebben die er is.

De CPU bestuurt alles in het apparaat waarin het zich bevindt, en is dus onmisbaar.

Logisch dus dat je de snelste CPU wilt hebben, maar er één kiezen is toch erg lastig.

Dit komt doordat de snelheid van een CPU door veel aspecten beïnvloed wordt,

Al (Of in ieder geval een groot deel ervan) deze aspecten willen wij aan jullie uitleggen.

De kloksnelheid

Als je ooit CPU's hebt bekeken, heb je deze term vast wel een keer gehoord

Het is namelijk één van de belangrijkste kenmerken voor de snelheid van een CPU

De kloksnelheid houdt namelijk in hoeveel berekeningen de CPU kan doen per seconde

Deze berekeningen besturen het hele apparaat, en zijn dus onmisbaar

Dus hoe meer berekeningen per seconde gedaan kunnen worden,

des te sneller een apparaat werkt

1?..2?...of toch 4?.

In elke CPU zit een bepaalde hoeveelheid 'cores'

Elke core wordt aangezien, en is eigenlijk ook, een aparte processor

Deze processoren kunnen individueel, of samen werken

Dus voor elke core extra in een CPU kan de computer meer
